# What is fileasy?

- png などの画像のリサイズや圧縮も可能です。
- base64 から png への変換
- svg から png や jpeg への変換
- pdf から png への変換

<br/>
重量課金の firebase や Azuru などのストレージに保存する前に、変換することができます。

## Install

```bash
npm install fileasy
```

of

```bash
npm i fileasy
```
